# Admission Form Web App

This is a simple and responsive **Admission Form** web application built using **React, HTML, and CSS**.  
The purpose of this project is to collect student information through a clean and user-friendly form interface.

## Features
- Student admission form
- Input fields for personal details
- Responsive design
- Easy to use UI
- Built with reusable React components
- Form validation (basic)

## Technologies Used
- React
- HTML5
- CSS3
- JavaScript

## Project Preview
This project displays an admission form where users can enter their details such as name, email, phone number, and other required information.
